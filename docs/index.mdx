# Blockstack Design System

## Principles

- **Is Always Extensible**
- **Be Consistent**
- **Be Flexible**
- **Do one thing well**
- **Do not prescribe**
- **Adapt and Improve**

#### With Yarn
```bash
yarn add blockstack-ui
```

#### With NPM
```bash

```
