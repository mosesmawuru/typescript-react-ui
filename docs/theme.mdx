# Theme

### Usage

You can import the theme into your app like so:

```jsx
// App.js

import { ThemeProvider } from 'styled-components'
import { theme } from '@blockstack/ui'

...

<ThemeProvider theme={theme}>
   // App goes here
</ThemeProvider>
```
---

## Colors


```jsx
const colors = {
  blue: Object.assign('#3700FF', {
    light: '#F1F6F9', // canvas
    mid: '#C4D8E5', // borders / labels on white
    neutral: '#403F83', // text
    dark: '#211F6D', // large fills
    accent: '#01FEFE' // cyan / accents / links
  })
}
```

<ColorComponent />

### Usage

```.jsx
<Box bg="blue.dark" size="200px" />
```

## Breakpoints
```jsx
const breakpoints = ['40em', '52em', '64em']
```
Nearly every prop on our components that map to a css propery can take an array. Each index of the array will be used for the corresponding breakpoint in this array.
### Usage
```.jsx
<Card width={['100%', '200px', '400px']}>
    Card Content
</Card>
```

## Box Shadows

```jsx
const shadows = {
  button: Object.assign('0px 2px 6px rgba(37, 0, 105, 0.22)', {
    hover: '0px 5px 9px rgba(37,0,105,0.28)',
    active: '0px 2px 2px rgba(37, 0, 105, 0.42)'
  }),
  card: Object.assign('0px 4px 4px rgba(0, 0, 0, 0.05)', {}),
  general: Object.assign('0px 4px 4px rgba(0, 0, 0, 0.05)', {})
}
```
### Usage
```.jsx
// box-shadow: 0px 5px 9px rgba(37,0,105,0.28)
<Box boxShadow="button.hover" bg="blue.dark" size="100px"  />
```
